# 📬 Contact Us Form

A clean, responsive contact form UI built with pure HTML and CSS — featuring a 
two-panel layout with contact details on the left and a message form on the right.

![HTML](https://img.shields.io/badge/HTML-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS-1572B6?style=flat&logo=css3&logoColor=white)
![Font Awesome](https://img.shields.io/badge/Font_Awesome-528DD7?style=flat&logo=fontawesome&logoColor=white)
![Google Fonts](https://img.shields.io/badge/Google_Fonts-4285F4?style=flat&logo=google&logoColor=white)

## Features
- Two-panel layout: contact info on the left, message form on the right
- Font Awesome icons for address, phone, and email contact details
- Name, email, and message input fields with styled focus states
- Hover animation on the Send button with smooth color transition
- Vertical divider between panels built with a CSS pseudo-element
- Fully responsive — stacks into a single column on mobile using media queries
- Google Fonts Poppins for clean, modern typography
- No JavaScript, no frameworks — pure HTML and CSS

## Tech Stack
| Layer | Technology |
|-------|------------|
| Structure | HTML5 |
| Styling | CSS3 (Flexbox, media queries) |
| Icons | Font Awesome 5 |
| Typography | Google Fonts Poppins |

## Setup
```bash
git clone https://github.com/Zemoik/Contact-Us-Form.git
cd Contact-Us-Form
open index.html
```
No dependencies, no build step, no internet connection required after clone.

## Project Structure
```
Contact-Us-Form/
├── index.html      # Markup — two-panel layout, contact info, form fields
├── styles.css      # All styling — layout, responsiveness, hover effects
└── README.md
```

## What I Learned
- Building multi-panel layouts using CSS Flexbox
- Creating a vertical divider using CSS `::before` pseudo-elements
- Writing responsive breakpoints with media queries to reflow layout on mobile
- Styling form inputs and buttons including hover and focus states
- Integrating Font Awesome icons via CDN for visual contact detail indicators
